Vote Eligibility Checking System
PROJECT OVERVIEW
The Vote Eligibility Checking System is a project designed to determine whether a person is eligible to vote based on age.
In India, the minimum age required for voting is 18 years.
PROJECT OBJECTIVE
-	Understand conditional logic
-	Work with user input
-	Calculate age
-	Apply real-world rules using code
TECHNOLOGIES USED
-	Python
-	Jupyter Notebook / VS Code- GitHub
FUNCTIONALITY
-	Accepts date of birth or age as input
-	Calculates age
-	Displays eligibility result
-	Provides user-friendly output
RULE SOURCE
Election Commission of India states that the legal voting age is 18 years.
HOW TO RUN THE PROJECT
1.	Download the project from GitHub
2.	Open the Python/Jupyter file
3.	Run the program
4.	Enter age or date of birth
SAMPLE OUTPUT
ELIGIBLE:
Congratulations! You are eligible to vote.
NOT ELIGIBLE:
Sorry! You are not eligible to vote yet.
